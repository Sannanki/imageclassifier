# Image Classification System

## 概要

ResNet-18（ImageNet事前学習済み）をベースとした，ブラウザで動作する画像分類Webアプリケーションです．
バックエンドは FastAPI（Python），フロントエンドはバニラ JS の SPA で構成されており，
クラス名の設定から訓練画像のアップロード・モデル学習・予測・可視化まで，コードを書かずにすべての操作をGUI上で完結できます．

## 技術的特徴

- **転移学習**: ImageNet で事前学習済みの ResNet-18 を使用し，少数の訓練画像でも高精度な分類を実現
- **2種類の学習方式**: 全層ファインチューニング（高精度）と最終層のみ更新する線形プローブ（高速・過学習しにくい）を選択可能
- **データ拡張**: ランダムクロップ・左右反転・色変動・回転を毎エポック自動適用し，少枚数でも過学習を抑制
- **差分学習率**: 全層学習時は事前学習済み Backbone を小さい学習率（`1e-4`），FC 層を大きい学習率（`1e-3`）で別々に最適化
- **CosineAnnealing スケジューラ**: 学習率をエポック終盤に向けてなめらかに減衰させ，収束精度を向上
- **クラス不均衡補正**: クラスごとの枚数比に応じて損失を自動で重みづけし，枚数差があっても精度を維持
- **可変クラス数**: 2クラス以上を自由に設定でき，クラスごとに異なる枚数の訓練画像に対応
- **ドラッグ＆ドロップ対応**: 訓練画像・予測画像・モデルファイルをドラッグ＆ドロップで操作可能
- **複数ファイル一括予測**: 複数の画像を同時にアップロードして一括推論し，結果をページ遷移後も保持（確率はクラス設定順で表示）
- **INT8動的量子化**: 学習後に自動でCPU推論専用の量子化モデルを生成し，推論を高速化
- **Gradient-based Saliency Map**: 予測に影響した画像領域をヒートマップで可視化（`autograd.grad` でモデルパラメータの勾配を計算せず入力勾配のみ取得することで高速化）
- **モデルの保存・再利用**: 学習済みモデルをクラス名情報ごと `.pth` ファイルに保存し，次回以降は再学習なしで即利用可能
- **リアルタイム進捗表示**: 学習の Epoch・Loss をリアルタイムでブラウザに表示
- **起動待機画面**: サーバー起動中はブラウザにスピナーを表示し，準備完了後に自動リダイレクト
- **マルチプラットフォーム対応**: Windows（PowerShell）・Mac・Linux に対応した起動スクリプトを同梱
- **GPU自動選択**: CUDA（Windows/Linux）→ MPS（Apple Silicon）→ CPU の優先順で自動的に最適なデバイスを選択
- **依存ライブラリ最小化**: PIL と NumPy のみで Saliency Map を描画
